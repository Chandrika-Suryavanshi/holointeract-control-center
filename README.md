## Installation

Clone the repository

```bash
git clone https://github.com/Chandrika-Suryavanshi/holointeract-control-center.git
```

Go to project directory

```bash
cd holointeract-control-center
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```
